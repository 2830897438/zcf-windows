# 呆呆鸟 API - Claude Code 配置工具

一键配置 [Claude Code](https://claude.ai/code) 使用呆呆鸟 API 中转服务。

站点：https://api.daidaibird.top

## 使用方法

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/2830897438/zcf-windows/main/setup.ps1 | iex
```

或下载后运行：

```powershell
powershell -ExecutionPolicy Bypass -File setup.ps1
```

### Linux / macOS (Bash)

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/2830897438/zcf-windows/main/setup.sh)
```
